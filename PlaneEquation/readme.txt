============================
PointHeight.java

-----
Compilation:

$ javac PointHeight.java

-----
Execution: the application takes a file name as an argument
which holds the 4 quad (x,y,z) points and the (x,z) point 
that requires a y- height.

e.g.

$ java PointHeight coordsH2.txt
